Правила игры

  1. Играем в закрытую. Не показываем карточки, даже после смерти.
  2. Бережно относимся к карточкам.
  3. Если Мафия и Маньяк стреляют друг в друга, то выстрел маньяка приоритетнее и убивает мафию, а мафия его нет.
  4. Блокираторт блокирует любое ночное действие, не может блокировать игрока две ночь подрят (только через ночь).
  5. Доктор может лечить любого игрока, но не может лечить две ночи подрят (только через ночь), себя может лечить только один раз за игру.
  6. Любовница дает имунитет один раз за игру любому игроку.
